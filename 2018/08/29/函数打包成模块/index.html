函数转化成模块

打包方法

将所需要打包的函数放到一个文件里，后缀为.py

使用方法

第一种情况是将函数写在文件里之后，在文件打开的状态下（在IDLE里打开）按F5，会调出运行环境，直接可以使用模块，此时函数前面不需要加命名空间。

IDLE是什么？
这个地方我弄得时候闹了个笑话，我把文件保存下来之后直接在运行的黑框框里按F5，可想而知的并没有什么卵用，后来查了IDLE的定义才懂
IDLE：是一个纯 Python 下自带的简洁的集成开发环境(IDE)，具备基本的IDE的功能，人家是编辑py代码的，意思就是先在这个IDLE里把你写的函数写好，保存成一个py文件，用IDLE打开，按F5，之后就可以直接用了
注意:直接用IDLE写函数按F5是不行的，他不是文件，也就不是模块（就没法弹出黑框框），必须保存成文件打开才可以
得到的知识:py的语言环境安装好之后，有一个语言运行环境的程序，还有一个IDLE

另一种情况是打开py的运行环境：
1、import 文件名（不需要加.py）
此时相当于打开该模块的命名空间，在当前空间里可以使用其中的函数
使用时前面要加模块名，如nester.foo()
2、from 文件名（不需要加.py） import 函数名
此时相当于把该模块下的函数加到当前空间里
使用时不需要加模块名，如foo()

将模块上传到PYPI

1、登录网站，注册账号
2、打开cmd（win+R打开运行，输入cmd）
先用cd （所在文件夹路径名）此条命令，打开所在文件夹
如cd C:\Users\lenovo\Desktop\test_upload
输入py setup.py sdist
此时文件夹里会出现MANIFEST文件以及一个叫dist的文件夹
输入py setup.py install
此时会出现build文件夹，此时模块已经准备好要上传了
